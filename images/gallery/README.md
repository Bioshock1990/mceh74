# Instagram Post Downloader

Программа для скачивания фотографий и описаний из постов Instagram по списку ссылок.

## Функциональность

- Скачивает все изображения из постов Instagram
- Сохраняет текст описания поста
- Создает отдельную папку для каждого поста
- Обрабатывает ошибки (удаленные/приватные посты)
- Добавляет задержку между запросами
- Показывает прогресс в консоли

## Установка

### 1. Установите Python 3.7+

### 2. Установите зависимости:

```bash
pip install -r requirements.txt
```

или

```bash
pip install instaloader
```

## Использование

### 1. Подготовка

Создайте файл `links.txt` в той же директории, где находится скрипт. Добавьте в него ссылки на посты Instagram (одна ссылка на строку).

Пример содержимого `links.txt`:

```
https://www.instagram.com/p/ABC123DEF/
https://www.instagram.com/p/XYZ789GHI/
https://www.instagram.com/reel/JKLM456NOP/
```

### 2. Запуск

Базовый запуск:
```bash
python instagram_downloader.py
```

С настройками:
```bash
python instagram_downloader.py --delay 3 --file my_links.txt
```

### Параметры командной строки

- `--delay N` - Задержка между запросами в секундах (по умолчанию: 2)
- `--file FILENAME` - Имя файла с ссылками (по умолчанию: links.txt)

## Структура выходных папок

Для каждого поста создается отдельная папка с именем `post_XXX_YYYYYYYY`, где:
- `XXX` - порядковый номер поста (001, 002, ...)
- `YYYYYYYY` - первые 8 символов ID поста

Внутри каждой папки:
- `image_1.jpg`, `image_2.jpg`, ... - изображения поста
- `description.txt` - файл с описанием поста

### Пример структуры:

```
post_001_ABC123DE/
├── image_1.jpg
├── image_2.jpg
└── description.txt

post_002_XYZ789GH/
├── image_1.jpg
└── description.txt
```

## Формат файла description.txt

```
Автор: @username
Дата: 2024-01-01 12:00:00
Лайков: 1234
Комментариев: 56
URL: https://www.instagram.com/p/ABC123DEF/

==================================================

Текст поста здесь...
```

## Обработка ошибок

Программа автоматически обрабатывает следующие ситуации:
- Пост удален или не существует
- Приватный профиль (требуется подписка)
- Ошибки сети
- Некорректные URL

В случае ошибки пост пропускается и программа продолжает работу со следующими ссылками.

## Важные замечания

1. **Задержка между запросами**: Instagram может ограничивать частые запросы. Рекомендуется использовать задержку не менее 2 секунд.

2. **Приватные профили**: Для скачивания постов из приватных профилей может потребоваться вход в аккаунт Instagram.

3. **Авторские права**: Используйте программу только для личного пользования и уважайте авторские права.

4. **Стабильность**: Библиотека instaloader может требовать обновлений при изменениях API Instagram.
